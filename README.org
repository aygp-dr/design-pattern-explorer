* Overview

A terminal UI (TUI) for interactively exploring Gang of Four design patterns. Built with [[https://github.com/charmbracelet/bubbletea][Bubble Tea]] and [[https://github.com/charmbracelet/lipgloss][Lip Gloss]].

* Install

#+begin_src bash
go install github.com/aygp-dr/design-pattern-explorer@latest
#+end_src

Or build from source:

#+begin_src bash
git clone https://github.com/aygp-dr/design-pattern-explorer.git
cd design-pattern-explorer
make build
./bin/design-pattern-explorer
#+end_src

* Controls

| Key              | Action                                        |
|------------------+-----------------------------------------------|
| =j= / =k=       | Navigate up/down                              |
| =Up= / =Down=   | Navigate up/down                              |
| =Enter= / =Space= | Toggle detail view                          |
| =1=              | Show all patterns                             |
| =2=              | Filter: Creational                            |
| =3=              | Filter: Structural                            |
| =4=              | Filter: Behavioral                            |
| =q=              | Quit                                          |

* Patterns Included

** Creational (5)
- Singleton
- Factory Method
- Abstract Factory
- Builder
- Prototype

** Structural (5)
- Adapter
- Bridge
- Composite
- Decorator
- Facade

** Behavioral (5)
- Observer
- Strategy
- Command
- State
- Template Method
